# LEP Community Intelligence Platform v2.0

Enterprise Analytics for Language Access & Health Equity

## Features

- **Executive Actions Dashboard** - Consolidated action items from all intelligence sources
- **Political & Regulatory Intelligence** - Drillable with actions, risks, and resources
- **Geographic Risk Assessment** - State-by-state with micro-location analysis
- **Social Determinants of Health** - 4-way comparison (English, LEP, D/HH, Blind)
- **Cultural Community Intelligence** - Health considerations and service gaps
- **Global Displacement Alerts** - UNHCR-sourced refugee movement data

## Tech Stack

- React 18
- Tailwind CSS
- Recharts
- Lucide React Icons
- Vite

## Getting Started

```bash
npm install
npm run dev
```

## Build
