# LEP Community Intelligence Platform

Enterprise Analytics for Language Access & Health Equity. Tracks Political, Geographic, Social, Cultural, and Economic dimensions with real-time monitoring of legislation, regulations, and best practices.

## Tech Stack

- React 18
- Vite
- Tailwind CSS v4
- Recharts
- Lucide React Icons

## Development

```bash
npm install
npm run dev
```

## Deployment

This project is configured for Vercel deployment. Connect the GitHub repo to Vercel and it will auto-detect the Vite framework.
